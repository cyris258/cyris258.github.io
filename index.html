底层const和顶层const

1. int const* ptr 或 const int* ptr **顶层 const：ptr 是一个指向 const int 的指针，const 位于类型（int）后面，表示指针指向的内容是常量，即底层 const**。 **底层 const**：指针 ptr 本身是可以修改的，所以这里没有顶层 const。 总结： **顶层 const**：无 **底层 const**：指向的 int 值是常量。 2. int* const ptr 顶层 const：ptr 是常量指针，const 位于指针（ptr）后面，表示指针本身是常量，不能修改指向的地址。即**顶层 const**。
