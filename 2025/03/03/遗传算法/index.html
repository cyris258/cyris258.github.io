遗传算法

2025-03-03 2025-03-05

遗传算法属于进化算法的一种，适用于解决针对某一场景下的优化问题。

关键字： 基因 染色体 个体 种群 遗传 变异 适应度

​ 在该算法思想中个体(individual)是由他的基因代表的，个体的基因绝大部分遗传于双亲，小部分由变异形成。

种群（population）则是由多个个体组成，每次种群的迭代都会产生新的个体从而形成新的种群。在其过程中双亲的交叉遗传尤为重要，

在此基础上种群的基因才能重新排列组合以实质上产生新的个体，即提供解决问题的一个新解。

算法实现

1.个体染色体的定义（一般只有一条染色体）

有如下几种形式：

表示形式 示例 适用问题
二进制编码 [1, 0, 1, 0, 1, 1, 0, 0] 离散优化问题（TSP、背包问题等）
实数编码 [3.5, 7.2, 2.8, 5.1] 连续优化问题
排列编码 [2, 0, 1, 3] 排列优化问题（TSP）
整数编码 [1, 3, 2, 4] 离散决策问题（背包问题等）
符号编码 ['A', 'B', 'C', 'D'] 非数值型问题（字符串匹配等）
树形编码 (add (x, 2) (multiply 3, y)) 遗传编程（函数优化）
分组编码 [[1, 2], [3, 4], [5]] 聚类、分配问题
多值编码 [3, 1, 2, 0] 调度问题、资源分配问题
混合编码 [1, 0, 3.5, 2.7, 0, 1] 复杂的组合优化问题

2.适应度函数定义，一般和染色体定义关联以求更加精确的求解某个问题的解。

3.其他行为定义，如交叉行为，变异，选择双亲，迭代等。

4.参数设置 ，如变异概率等。
